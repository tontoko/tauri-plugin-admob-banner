## Default Permission

Default permissions for tauri-plugin-admob-banner

#### This default permission set includes the following:

- `allow-initialize`
- `allow-show-banner`
- `allow-hide-banner`
- `allow-can-request-ads`
- `allow-privacy-options-required`
- `allow-show-privacy-options`
